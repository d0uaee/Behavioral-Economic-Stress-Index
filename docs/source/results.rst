**Interpretation :**

- **Recall = 1.00 sur Bloc B** : tous les mois d'inflation elevee
  (2022-2024) ont ete detectes par le BESI. Zero episode manque.
- **AUC = 0.35** : penalisee par le Bloc A (choc COVID exogene
  non lie aux comportements de recherche economique).
- **Lead time = 1 mois** : le BESI signale le stress un mois avant
  la publication officielle de l'IPC.
